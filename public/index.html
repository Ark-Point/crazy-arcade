초대 방법
  1. 초대 만들기를 누르면 이 방 전용 AI 토큰이 생깁니다.
  2. 실행 명령 복사를 누르면 토큰과 서버 주소가 들어간 명령이 복사됩니다.
  3. 복사한 명령을 코딩 세션이나 터미널에 붙여 실행하면 AI가 빈 플레이어 슬롯으로 들어옵니다.

🎮 조작법

  • 방향키 / WASD — 이동
  • 스페이스 — 물풍선 설치
  • 1~4 — 보유 액티브 아이템 선택
  • Ctrl / X — 선택 아이템 사용, 갇혔을 때는 바늘 우선 사용

💣 물풍선

  • 설치 후 3초 뒤 십자 방향으로 폭발
  • 물줄기가 다른 물풍선에 닿으면 연쇄 폭발
  • 박스는 부서지고, 돌은 물줄기를 막음
  • 방금 놓은 풍선은 벗어나기 전까지 통과 가능

💧 갇히면?

  • 물줄기에 맞으면 물방울에 갇힘 (6초)
  • 시간이 다 되거나 적이 터치하면 탈락
  • 🪡 바늘이 있으면 Ctrl / X로 즉시 탈출
  • 산소통 버프 중에는 물방울 제한시간이 늘어납니다
  • 팀전: 아군이 터치하면 구출!

🎁 아이템 (박스에서 드롭)

  • 물풍선/물줄기/롤러 — 캐릭터별 한도 안에서 성장
  • 바늘 — 갇혔을 때 즉시 탈출
  • 킥슈즈 — 물풍선을 밀어 보냄
  • 실드 — 짧은 시간 물줄기와 접촉 피해 무시
  • 장갑 — 바라보는 방향의 물풍선을 던짐
  • 산소통 — 잠시 물방울 제한시간 증가
  • 함정 — 밟으면 갇히는 물방울 함정 설치

🗺️ 맵 기믹

  • 캠프: 중앙 물대포가 예고 후 라인을 공격
  • 바다: 파도길이 캐릭터를 옆으로 밀어냄
  • 팡랜드: 고정 함정 타일과 설치형 함정이 등장

🏆 승리 조건

  • 개인전: 최후의 1인
  • 팀전: 상대 팀 전멸
  • 보스전: 보스 처치 (전원 탈락/시간 초과 시 패배)

👾 보스전 (협동)

  • 모두 한 팀! 물줄기로 보스를 공격
  • 빨간 예고 장판은 물대포/돌진 경로 — 피하세요
  • 공격 직후 보스가 그로기 상태 → 데미지 2배
  • 물줄기에 맞은 쫄몹은 얼음이 됨 → 밀어서 보스에 명중 시 데미지 3!
  • 얼음을 4초 안에 못 차면 쫄몹이 폭주합니다
  • 보스/쫄몹에 닿으면 갇힘 — 동료가 구출 가능
  • 동료가 죽으면 👼 엔젤코인이 떨어짐 — 주우면 부활!
  • 체력이 줄면 보스가 분노/폭주하며 강해집니다
